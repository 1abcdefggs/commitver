# Test Project

This is a simple test project to verify COMMITVER workspace functionality.

## Purpose
- Test workspace dependency resolution
- Verify @commitver/core functionality
- Demonstrate monorepo integration

## Usage
```bash
# From the root of the monorepo
npm run test-project

# Or directly from test project
cd test-project
npm start
```

## Expected Output
When you run `npm start`, you should see:
- Version formatting test results
- Confirmation that COMMITVER core is working
- Workspace integration verification
